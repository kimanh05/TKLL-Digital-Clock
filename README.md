# Digital Clock (Verilog - Xilinx Vivado)

## Overview
A Verilog-based digital clock implemented in Xilinx Vivado with modular and scalable design.

## Features
- Four-layer module hierarchy for modular design.
- Functions: time setting, stopwatch, countdown, and clock divider.

## Tools
- Xilinx Vivado

## Result
- Successfully simulated and synthesized the clock in Vivado.
